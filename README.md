# TasteFit

A Flutter recipe and nutrition tracking app with Firebase backend integration.

## Features

- **Authentication**: Firebase email/password sign-in and registration
- **Recipe Discovery**: Browse recipes by category with rich imagery and nutrition info
- **Recipe Details**: View ingredients, cooking steps, nutrition facts, and ratings
- **Search**: Full-text recipe search with suggestions
- **Favorites**: Save and manage favorite recipes
- **Shopping List**: Create categorized shopping lists with quantity/unit tracking
- **Nutrition Tracker**: Log meals and track daily macronutrients
- **Multi-Platform**: Runs on mobile and web with shared codebase

## Tech Stack

- **Framework**: Flutter
- **State Management**: Riverpod
- **Backend**: Firebase (Firestore, Auth, Analytics, Crashlytics)
- **Platforms**: iOS, Android, Web

## Getting Started

1. Clone the repository
2. Run `flutter pub get` to install dependencies
3. Configure Firebase (see `firebase_options.dart`)
4. Run `flutter run` to launch the app
