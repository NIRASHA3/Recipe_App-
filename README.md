# 🥗 TasteFit – Recipe & Nutrition Tracker

A **Flutter-based recipe and nutrition tracking app** that helps users discover recipes, manage shopping lists, and track daily nutrition.  
Integrated with **Firebase** for authentication, data storage, and analytics.

## ✨ Features

- 🔐 **Authentication** – Firebase email/password sign-in and registration  
- 🍲 **Recipe Discovery** – Browse recipes by category with images and nutrition info  
- 📖 **Recipe Details** – View ingredients, cooking steps, nutrition facts, and ratings  
- 🔎 **Search** – Full-text recipe search with suggestions  
- ❤️ **Favorites** – Save and manage favorite recipes  
- 🛒 **Shopping List** – Categorized shopping lists with quantity/unit tracking  
- 📊 **Nutrition Tracker** – Log meals and track daily macronutrients  
- 📱 **Multi-Platform** – Runs on mobile and web with shared codebase

## ⚙️ Getting Started

Follow these steps to run TasteFit on your machine:

```bash
1. Clone the repository:
git clone https://github.com/NIRASHA3/Recipe_App.git

2. Navigate to the project folder:
cd tastefit_latest

3. Install dependencies using Flutter:
flutter pub get

4. Configure Firebase:
Make sure your firebase_options.dart is properly set up with your Firebase project.

5. Run the app on an emulator or physical device:
flutter run
